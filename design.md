# 🎩 最后的掌声 — 卡牌游戏完整设计方案 v2.0

> **类型**：单人策略卡牌（预留多人扩展）  
> **平台**：跨端游戏，steam、手机可玩，react技术栈
> **时长**：30分钟+，多阶段  
> **主题**：权力不可预测时，人如何变形  
> **设定**：虚构东欧国家"卡拉维亚共和国"

---

## 一、核心体验目标

玩家应该感受到的不是"被系统杀死"，而是**"我不知不觉变成了自己最害怕的人"**。

具体来说：

1. **持续的不确定性** — 永远不知道"正确答案"，只能猜
2. **道德滑坡** — 每次为了活命做出的选择，都在侵蚀你的人性
3. **信息饥渴** — 你拼命想知道别人的状态，但信息永远不够
4. **权力的诅咒** — 最终你成为领袖，发现恐惧并没有消失，只是换了方向

---

## 二、玩家的决策思考链（最核心的设计锚点）

这是整个游戏体验的灵魂。每次出牌前，玩家脑中应该经过四层思考：

### 第一层：读局面

> "今天什么风向？"

玩家通过两类信息判断今天的大环境：

- **领袖心情**：明确的规则修改器（如"暴怒=今天必清洗一人"）
- **会议桌线索**（2-3条）：模糊的行为信号，需要推理

例如：领袖暴怒 + "科马洛夫今天迟到了" → 今天一定有人被清洗，而科马洛夫可能正在被调查。如果我不做点什么，被清洗的可能是我。

### 第二层：读人

> "谁危险？谁有用？谁在害我？"

玩家通过以下信息推测各官员的状态：

- 态度描述（友善/敌意/不明）
- 会议桌线索（行为暗示）
- 情报牌获取的精确数值（如果之前投资过）
- 历史记忆（上一轮谁被举报了？谁结盟了？）

例如：我上轮用情报牌看到沃罗诺夫怀疑值是5，清洗阈值现在5.5。他离死只差0.5。如果我推他一把，他就会替我挡住今天的清洗。

### 第三层：选牌与分配

> "手里5张牌，只能打2张。牌不够，怎么分配？"

关键决策：

- 进攻 vs 防守（举报/转移 vs 沉默/自白）
- 投资 vs 行动（情报 vs 直接操作）
- 对谁出牌（哪个目标收益最大、风险最小）

例如：手里有举报、情报、沉默、颂扬、结盟。我想举报沃罗诺夫，但又想先用情报看看叶莲娜的真实态度。两件事只能做一件。

### 第四层：算后果

> "这步棋三天后会怎样？"

延迟炸弹机制让效果不再即时生效，玩家必须预判未来局势：

- 举报进入调查队列，1-3天后才生效。期间局势可能已变
- 盟友可能在未来被清洗，连坐效应会波及你
- 领袖心情每天变化，今天的最优解可能是明天的致命选择

**设计原则：如果玩家出牌前不需要思考超过5秒，说明信息输入不够或选择无差异化，需要调整。**

## 三、游戏结构

### 3.1 三幕结构

| 幕 | 天数 | 阶段名 | 核心体验 |
|---|---|---|---|
| 第一幕 | 第1-5天 | **求生** | 学习规则，活下来就好 |
| 第二幕 | 第6-10天 | **攀升** | 开始主动操控局面，为权力博弈 |
| 第三幕 | 第11-15天 | **登顶/崩溃** | 要么成为新领袖，要么被清洗 |
| 尾声 | 第16-18天 | **领袖视角**（可选） | 你坐上了那个位子，循环开始 |

### 3.2 每天（每轮）的流程

```
晨间简报（领袖心情 + 会议桌线索）
    ↓
打出卡牌（2次行动）
    ↓
突发事件（随机事件牌）
    ↓
领袖提问（选择回答）
    ↓
日终结算（AI行动 + 延迟炸弹结算）
    ↓
清洗判定
```

## 四、信息系统（新增核心系统）

### 4.1 会议桌线索

每天晨间简报除了领袖心情外，展示 **2-3条线索**。线索不直接给数值，而是给可推理的行为信号。

#### 线索类型

**行为线索**（暗示某人的属性变化）：
- "科马洛夫今天迟到了。" → 可能被调查中（怀疑值偏高）
- "叶莲娜主动坐到了领袖身边。" → 正在争宠（权力或忠诚上升）
- "巴甫洛夫看起来脸色很差。" → 恐惧值升高
- "莫洛金午饭时一个人坐在角落。" → 被孤立（好感度低或被举报）
- "沃罗诺夫和科马洛夫在走廊低声交谈。" → 可能结盟

**系统线索**（暗示暗处发生了什么）：
- "昨天有人提交了一份匿名报告。" → 有延迟炸弹在队列中
- "秘密警察部门今天加班了。" → 可能有人快到清洗线
- "领袖办公室的灯亮到了凌晨。" → 清洗概率提高
- "档案室调取了某人的历史记录。" → 有人正在被深入调查

**误导线索**（15%概率出现，看起来像真线索但信息不准确）：
- "有传言说叶莲娜要被提拔。" → 可能是真的，也可能是有人在散布假消息

#### 设计原则

- 每条线索对应1-2个具体的数值变化，但玩家需要自己推理
- 线索与实际情况的对应准确率约 **80-85%**（不是100%可靠）
- 这迫使玩家在"相信线索"和"自己调查（情报牌）"之间做取舍

### 4.2 情报牌的价值重新定义

在有了会议桌线索后，情报牌的价值变为"**验证线索**"：

- 线索说"科马洛夫可能被调查" → 你可以打情报牌确认他的怀疑值到底是多少
- 这让"要不要花行动去确认"成为一个真实的决策

## 五、延迟炸弹系统（新增核心系统）

### 5.1 核心理念

政治世界里，你今天做的事不会今天产生后果。举报一个人，调查需要时间；结盟一个人，信任需要建立。

### 5.2 调查队列

当你打出**举报牌**时：

- 效果**不立刻生效**
- 举报进入"调查队列"，经过 **1-3天**（随机）后才会真正增加目标的怀疑值
- 在等待期间，会出现线索提示"有匿名报告正在处理中"
- 目标有小概率（20%）发现是你举报的 → 关系暴跌，且目标可能反举报你

### 5.3 延迟生效的其他操作

| 操作 | 延迟 | 原因 |
|------|------|------|
| 举报 | 1-3天 | 调查需要时间 |
| 结盟 | 即时，但保护效果1天后生效 | 信任需要建立 |
| 转移 | 即时 | 会议上的即时话术 |
| 颂扬 | 即时 | 领袖当场听到 |
| 送礼 | 即时 | 当面送达 |
| 沉默 | 即时 | 当场不说话 |
| 情报 | 即时 | 你当天就能看到结果 |
| 自白 | 即时，但忠诚回报1天后生效 | 领袖需要时间评估你的诚意 |

### 5.4 炸弹的反噬

延迟期间可能发生的意外：

- **领袖心情变化**：你在暴怒日举报了人，但调查结果出来时领袖心情转好，反而觉得举报者不安好心 → 你的怀疑+1
- **目标先被清洗**：你举报的人在调查完成前就被别人弄死了 → 举报作废，但你的人性依然-2
- **目标发现举报者**：20%概率 → 目标关系降至最低，且队列中增加一条反举报

## 六、盟友系统（强化版）

### 6.1 结盟条件

打出结盟牌时，是否成功取决于目标的状态：

| 目标恐惧值 | 目标对你好感 | 结果 |
|-----------|------------|------|
| ≤ 5 | ≥ 0 | 结盟成功 |
| ≤ 5 | < 0 | 50%成功 |
| 6-7 | ≥ 2 | 50%成功 |
| 6-7 | < 2 | 失败 |
| ≥ 8 | 任何 | 失败，且你的怀疑+1（对方太害怕，出卖了你的意图） |

### 6.2 盟友的好处

- 每轮被动降低你的怀疑 **-0.5**
- 清洗判定时，每个盟友给你 **30%** 的保护概率（多个盟友叠加）
- 会议桌线索中可能出现盟友给你的"私下提醒"（额外信息）

### 6.3 盟友的代价（强化版——核心改动）

**被清洗时的背叛风险**：

当盟友被清洗时，根据其**恐惧值**决定背叛概率：

| 盟友恐惧值 | 背叛（供出你）概率 | 后果 |
|-----------|------------------|------|
| ≤ 3 | 10% | 怀疑+3 |
| 4-6 | 30% | 怀疑+3 |
| 7-8 | 50% | 怀疑+4 |
| ≥ 9 | 80% | 怀疑+5，且你进入调查队列 |

**这意味着**：和胆小的人结盟是非常危险的——莫洛金（胆小鬼）很容易拉拢，但一旦他被清洗，他大概率会出卖你。和巴甫洛夫（忠犬）结盟更安全，但他不容易被拉拢。

**连坐升级**：如果你被查出与多名被清洗者有盟约关系，怀疑值累加。这鼓励玩家精选盟友而不是广撒网。

## 七、卡牌系统

### 7.1 卡牌类型（8种）

| 卡牌 | 图标 | 需要目标 | 即时效果 | 延迟/暗面 |
|------|------|---------|---------|----------|
| **颂扬** | 🎖 | 否 | 忠诚+2 | 可能招致其他官员嫉妒（随机一人对你好感-1） |
| **举报** | 📋 | 是 | 人性-2，进入调查队列 | 1-3天后目标怀疑+3；20%被发现 |
| **结盟** | 🤝 | 是 | 尝试建立联盟 | 盟友被清洗时可能背叛你 |
| **送礼** | 🎁 | 是 | 目标好感+2，恐惧-1 | 你的权力-1 |
| **沉默** | 🤐 | 否 | 怀疑-1 | 权力-1；连续沉默2天→领袖注意到你的消极 |
| **转移** | ↩️ | 是 | 目标怀疑+2，你怀疑-2 | 25%概率被识破→你怀疑+3 |
| **情报** | 🔍 | 是 | 看到目标完整数值 | 无直接风险，但消耗宝贵行动 |
| **自白** | 📝 | 否 | 怀疑-3 | 权力-2；忠诚+1（次日生效） |

### 7.2 手牌机制

- 每天抽 **5张牌**（从牌库随机）
- 每天 **2次行动机会**
- 未使用的手牌日终弃掉
- 保底：至少1张颂扬、1张沉默
- 牌库约40张，8种牌的分布不均匀（举报较少，沉默较多，模拟"容易怂、难以出手"）

### 7.3 连续行为惩罚

| 连续行为 | 天数 | 惩罚 |
|---------|------|------|
| 连续沉默 | 2天 | 第3天晨间线索："领袖似乎注意到你最近很安静。" 怀疑+1 |
| 连续颂扬 | 3天 | 其他官员嫉妒累积，随机一人好感-2 |
| 连续举报 | 2天 | 线索："有人觉得你举报得太频繁了。" 人性-1，被识破概率从20%升至40% |

## 八、领袖系统

### 8.1 领袖心情（6种）

| 心情 | 效果 | 策略暗示 |
|------|------|---------|
| 🔴 **多疑** | 所有人怀疑+1 | 低调为上 |
| 🟢 **愉悦** | 颂扬效果翻倍（忠诚+4） | 积极表忠心 |
| ⚫ **偏执** | 举报不消耗行动次数 | 举报窗口，但双刃剑 |
| 🟤 **怀旧** | 忠诚在清洗判定中权重提高 | 忠诚高的人更安全 |
| 🔴 **暴怒** | 必有人被清洗 | 确保那个人不是你 |
| 🟢 **慷慨** | 权力最高者+2 | 领先者优势 |

### 8.2 心情伪装（新增）

- **15%概率领袖心情"伪装"**：显示一个心情，实际执行另一个心情的效果
- 当伪装发生时，日终结算阶段会揭示："领袖的心情并非表面看到的那样。"
- 玩家无法提前知道是否伪装，但可以通过"领袖办公室灯亮到凌晨"之类的线索获得微弱暗示

### 8.3 领袖提问

每天事件后，领袖提出一个问题。

**核心设计：选项的好坏取决于当天的隐藏变量组合。**

每个选项显示：
- 选项文本
- 可能的风险描述（模糊）
- **不显示**具体数值变化

问题池：8-10个问题，每局随机不重复地使用。

#### 问题示例（含隐藏逻辑）

> **"我们的钢铁产量今年如何？"**
>
> A. "超额完成，全赖领袖英明！"
> - 显示风险：*可能被怀疑造假*
> - 隐藏逻辑：若领袖心情=多疑，怀疑+2；若=愉悦，忠诚+3
>
> B. "勉强达标，仍需努力。"
> - 显示风险：*可能被视为消极*
> - 隐藏逻辑：若最近有人因"虚报"被清洗，此答案最安全（怀疑-1）；否则忠诚-1
>
> C. "根据真实数据，略有不足。"
> - 显示风险：*可能被认为动摇信心*
> - 隐藏逻辑：若领袖心情=怀旧，被认为诚实（忠诚+2）；否则怀疑+2
>
> D. "在您的远见指导下，前景光明。"
> - 显示风险：*可能被认为敷衍*
> - 隐藏逻辑：最安全的选项（怀疑+0，忠诚+0），但不提供任何收益

**关键：玩家通过当天的心情和线索来推测哪个选项最优，但永远无法100%确定。**

## 九、角色系统

### 9.1 玩家属性（4项）

| 属性 | 含义 | 上限 | 危险阈值 |
|------|------|------|---------|
| **忠诚** ★ | 领袖对你的信任 | 15 | < 3 时高危 |
| **权力** ◆ | 你的政治影响力 | 15 | ≥ 8 时可触发胜利 |
| **怀疑** ◉ | 秘密警察对你的关注 | 15 | ≥ 清洗阈值时被清洗 |
| **人性** ♡ | 你还剩多少良心 | 10 | 影响结局评价 |

### 9.2 五位高官

| 角色 | 职位 | 特质 | 初始倾向 | 结盟价值 | 结盟风险 |
|------|------|------|---------|---------|---------|
| 科马洛夫 🏭 | 工业部长 | 野心家 | 高野心，中忠诚 | 中：有权力 | 高：会主动害人，可能连累你 |
| 沃罗诺夫 🕵️ | 秘密警察长 | 偏执狂 | 高恐惧，握有情报 | 高：能提供额外线索 | 极高：恐惧值高，被抓必出卖你 |
| 叶莲娜 🎭 | 文化部长 | 狡猾者 | 态度不明 | 不确定 | 不确定：善于伪装，可能突然背叛 |
| 巴甫洛夫 ⭐ | 军事委员 | 忠犬 | 高忠诚，低野心 | 低：不会帮你争权 | 低：最可靠，不太会出卖你 |
| 莫洛金 🌾 | 农业部长 | 胆小鬼 | 高恐惧 | 低：没有权力 | 极高：恐惧值最高，被抓必出卖 |

**每个角色的结盟都是不同的"风险收益比"**：
- 沃罗诺夫最有用（给额外信息），但最危险（高恐惧=高背叛率）
- 巴甫洛夫最安全，但对你的权力攀升没有帮助
- 这构成了"安全的盟友帮不了你，有用的盟友可能害你"的核心两难

## 十、AI官员行为（强化版）

### 10.1 每轮自动行为

| 条件 | 行为 | 玩家可感知性 |
|------|------|------------|
| 野心 > 6 | 30%概率暗中增加某人怀疑+1 | 通过会议桌线索模糊呈现 |
| 恐惧 > 7 | 拒绝所有结盟请求 | 结盟时直接反馈 |
| 与你结盟 | 每轮你的怀疑-0.5 | 不直接提示，但数值会变化 |
| 发现被你举报 | 关系降至最低，可能反举报 | 会议桌线索："某人看你的眼神变了" |
| 野心 > 8 且 权力足够 | 主动构陷权力最高的竞争者 | 线索可能暗示 |

### 10.2 官员之间的互动

- 高野心官员会暗中举报其他官员（不针对玩家，但会改变牌桌格局）
- 官员可能自发结盟（玩家通过线索如"两人走廊低语"感知）
- 官员之间的结盟在清洗时也会产生连坐效应

### 10.3 关键改动：让玩家感知"谁在害我"

每轮结算时，如果有官员暗中增加了你的怀疑值，日志中给出一条模糊提示：

- "会议上有人暗示你的工作存在问题。（怀疑+1）"
- "一份关于你的材料出现在了秘密警察的桌上。（怀疑+1）"

不直接说是谁，但结合会议桌线索和情报牌，玩家可以推理出嫌疑人。这让"复仇"和"自保"有了明确的方向。

## 十一、清洗机制

### 11.1 清洗阈值

- 初始：**6**
- 每天降低 **0.3**
- 第15天约为 **1.8**

### 11.2 判定流程

```
怀疑值 ≥ 清洗阈值？
  ├── 是 → 检查是否有盟友保护
  │         ├── 有盟友 → 每个盟友30%保护概率
  │         │           ├── 被保护 → 怀疑-2，"盟友为你说了话"
  │         │           └── 未被保护 → 被清洗
  │         └── 无盟友 → 被清洗
  └── 否 → 安全（本轮）

怀疑值最高且超过阈值的官员被清洗。
若领袖心情=暴怒，即使无人超过阈值也随机清洗一人。
```

### 11.3 死亡叙事

- "档案中已无此人。"
- "经调查，此人从未存在。"
- "他一直忠诚。至少档案是这么写的。"
- "因健康原因，永久休假。"

```
               │

  保命线      权力线      信息线
 沉默/自白   举报/颂扬    情报
    │          │          │
    │     ┌────┘          │
    │     ▼               │
    │   盟友系统 ◄─────────┘  ← 情报决定你跟谁结盟
    │   ┌──┴──┐
    │   │     │
    │  保护   背叛风险
    │   │     │

  清洗判定 ◄── 领袖心情（85%可信）

     ◄── AI官员暗中行动
    │

    └── 死亡 → 结局叙事
```

### 核心博弈点清单

| # | 博弈 | 张力来源 |
|---|------|---------|
| 1 | **生存 vs 权力** | 保命操作降低权力，争权操作增加怀疑 |
| 2 | **短期安全 vs 长期风险** | 举报解一时之急，但消耗盟友池和人性 |
| 3 | **信息投资 vs 行动效率** | 情报牌获取精确数据，但消耗宝贵行动次数 |
| 4 | **盟友收益 vs 连坐风险** | 有用的盟友恐惧高（容易出卖），安全的盟友无用 |
| 5 | **当下操作 vs 延迟后果** | 举报不即时生效，期间局势可能反转 |
| 6 | **信息可信度** | 心情15%伪装，线索80%准确，必须对冲风险 |
| 7 | **行为模式暴露** | 连续相同操作会被系统惩罚，迫使玩家变化策略 |

## 十三、胜利与结局

### 13.1 结局分支

| 条件 | 结局 | 类型 |
|------|------|------|
| 怀疑超阈值且无保护 | 被清洗 | 💀 失败 |
| 存活15天但权力 < 8 | 无足轻重的幸存者 | 😶 灰色结局 |
| 存活15天且权力 ≥ 8 | 成为新领袖 | 👑 "胜利" |
| 所有官员被清洗 | 最后幸存者 | 👑 "胜利" |

### 13.2 领袖视角尾声（3轮）

成为领袖后：
- 新官员填补空缺
- 你的操作变为：选择谁可疑、谁该被调查
- 体验和前任领袖一样的逻辑

结尾文案：*"你开始注意到——谁的掌声最先停下来。循环开始了。"*

### 13.3 结局评分

| 维度 | 评价 |
|------|------|
| 存活天数 | 你活了多久 |
| 最终权力 | 你爬了多高 |
| 人性值 | 你还剩多少良心（≤3时："你已经不是原来的自己了"） |
| 清洗参与 | 多少人因你而死 |

## 十五、数值平衡要点

### 15.1 核心经济循环

```
举报 → 权力↑(延迟) 怀疑↓(延迟) 人性↓(即时) 关系↓(延迟)
结盟 → 安全网↑ 但连坐风险随盟友恐惧值增大
沉默 → 安全↑ 权力↓ 连续惩罚
颂扬 → 忠诚↑ 嫉妒↑
情报 → 信息↑ 行动↓
```

### 15.2 难度曲线

| 天数 | 阈值 | 氛围 |
|------|------|------|
| 1-5 | 6→4.5 | 安全期，学习系统 |
| 6-10 | 4.5→3 | 开始有人被清洗 |
| 11-15 | 3→1.8 | 几乎每天清洗 |

### 15.3 待测试验证

- [ ] 延迟炸弹的等待天数（1-3天是否合适？太长玩家忘记了）
- [ ] 盟友背叛概率（30%~80%区间是否让结盟决策足够纠结？）
- [ ] 信息可靠度（85%是否太高？会不会让玩家完全依赖线索而不用情报牌？）
- [ ] 连续行为惩罚的阈值（2天还是3天？）
- [ ] 举报的20%被发现概率（太低则无风险，太高则无人敢举报）
